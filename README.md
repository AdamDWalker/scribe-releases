# Scribe Releases

Public update feed for the [Scribe](https://github.com/AdamDWalker/scribe) macOS app (source is private).

- `appcast.xml` — the Sparkle update feed (served via raw.githubusercontent.com)
- Release assets — the signed app zips, attached to each GitHub Release here

Managed by `scripts/release.sh` in the private source repo. Do not edit by hand.
